Regras de Implementação

O que FAZER (Sempre)

  • Usar Playfair Display apenas para Display (H1/H2 e citações). Exige peso 500/600/700.
  • Aplicar `prefers-reduced-motion` no CSS global de toda a página para desativar as animações complexas se o usuário solicitar.
  • Manter área de clique mínima de 48x48px em botões mobile.
  • Intercalar seções claras com no máximo 1 ou 2 cards escuros (`.section-card.dark`) reservados para as maiores quebras de conversão (ex: depoimentos ou formulário).

O que NÃO FAZER (Nunca)

  • Nunca remover o `border-radius: 1.5rem` nem grudar a `.section-card` na borda da tela no Desktop. O fundo `#04152d` deve sempre aparecer.
  • Nunca usar cores vibrantes randômicas. A paleta é restrita: Azul Principal, Navy, Dourado e tons de cinza/branco.
  • Nunca aplicar a animação `gv-fade-slide-in` em todos os elementos da tela ao mesmo tempo. Use as classes `.delay-*` para fazer Staggering.
